# Mask2Former

Masked-attention Mask Transformer (Mask2Former) is a segmentation architecture capable of addressing any image segmentation task (panoptic, instance or semantic)

- **🔗 GitHub**: [https://github.com/facebookresearch/mask2former](https://github.com/facebookresearch/mask2former)
- **📄 Paper**: [https://doi.org/10.48550/arXiv.2112.01527](https://doi.org/10.48550/arXiv.2112.01527)
